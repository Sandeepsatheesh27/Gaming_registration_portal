Zombie Hunter Esports – Gaming Registration Portal
A simple, fast, and cool gaming registration website for tournaments in Valorant, Fortnite, and Marvel Rivals.

Players can:

Register with Name, Age, Email, Game, and IGN
View upcoming tournaments
Watch YouTube highlights
Join via Instagram & YouTube
Built with pure HTML + JavaScript (no backend needed), and containerized with Docker so anyone can run it in 1 second.
